# Development Phases & Roadmap

This roadmap is built for fast execution, validation, and short feedback loops.

---

# Phase 1 — Screenshot-Based Demo MVP (Core Value)
### Goal
Deliver core demo guidance using screenshot-based flows.

### Build:
- **Browser Extension**:
  - Screenshot capture during workflow recording
  - Click/navigation tracking
  - Screenshot upload to object storage
- **Web App**:
  - Flow creation with screenshot steps
  - Step navigation and viewing
  - Step titles, descriptions, and notes
  - Flow management (create, edit, delete)

### Outcome
A functioning prototype where users can record workflows via extension (capturing screenshots) and view screenshot-based demo flows in the web app.

---

# Phase 2 — Web App + Multi-Tenant SaaS Foundation
### Build:
- Next.js app
- Auth.js (email/password or Google)
- Organizations (create, switch)
- Memberships (OWNER, ADMIN, MEMBER)
- Flow CRUD (create, update, delete)
- Flow list + tagging
- Screenshot storage and management
- Extension integration:
  - Extension authentication
  - Screenshot upload API
  - Flow sync from extension

### Outcome
Your first "real" SaaS product. Ready for early adopters and pilots.

---

# Phase 3 — Branching, Rehearsal Mode & Versioning
### Build:
- Branching paths (goto step)
- Tree editor for branching demo flows
- Rehearsal/practice mode for reps
- Flow versioning & publishing

### Outcome
Teams can create professional demo playbooks.

---

# Phase 4 — Advanced Features
### Build:
- Flow templates and presets
- Step annotations and callouts
- Interactive hotspots on screenshots

### Outcome
Enhanced demo creation and presentation capabilities.

---

# Phase 5 — AI Layer
### Build:
- AI rewrite for steps/notes
- Persona-specific demo variations
- Generate flow summaries
- Generate example flows automatically

### Outcome
Less friction in creating + maintaining demo flows.

---

# Phase 6 — Analytics + Admin + Billing
### Build:
- Flow usage analytics
- Rep performance insights
- Demo session logs
- Stripe billing for Starter/Team/Scale plans

### Outcome
Full SaaS maturity. Ready for growth.
